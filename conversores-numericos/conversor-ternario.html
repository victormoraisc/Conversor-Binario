Como converter de Ternário (base 3) para decimal?

A Conversão de Ternário para Decimal é bem simples, basta inverter o número ternário e então multiplicar cada algarismo por uma potência de 3, começando com 3º e então somar todos os resultados para obter o número Decimal.

Exemplo:

Vamos converter o número Ternário 2012 para Decimal, primeiramente vamos invertê-lo e teremos 2102;

2*3º = 2
1*3¹ = 3
0*3² = 0
2*3³ = 54

Por fim, somaremos os resultados das multiplicações e teremos 59 como resultado.

Como converter de decimal para Ternário (base 3)?

Já para converter de Decimal para Ternário vamos dividir o número decimal por 3 e o resultado inteiro das divisões sucessivas por 3 até que o resultado seja menor que 3. Enfim usaremos o resto das divisões e o resultado da última divisão para construirmos o número Ternário.

Exemplo:

Vamos converter o número 59 para Ternário, portanto, primeiramente vamos realizar as divisões por 3.

59/3 = 19, Resto 2 (4º Número)
19/3 = 6, Resto 1(3º Número)
6/3 = 2, Resto 0(1º e 2º Número)

Enfim vamos montar o número utilizando primeiro o último resultado das divisões e após usaremos o último resto, e então utilizaremos os anteriores para montar o resultado, que é 2012.

Curiosidades Sobre o Sistema Ternário


A base ternária é compacta

Apesar de ter uma base pequena os números ternários não aumentam a quantidade de dígitos em uma medida astronômica como os binários, devido a essa propriedade eles são extremamente compactos levando em conta o tamanho da base.

Computação Ternária

Na computação existiram computadores ternários, que eram baseados em lógica ternária ao invés dos convencionais com lógica binária, e ao contrario dos irmãos binários os computadores ternários possuíam trits no lugar de bits e com apenas 18 trits um computador ternário conseguiria operar uma variação numérica equivalente a 29 bits da sua contraparte binária. Apesar da maior eficiência os processadores ternários nunca foram explorados à fundo por conta do seu alto custo de produção e de desenvolvimento.

É possível contar até 59.048 com as mãos utilizando o sistema ternário

Normalmente contamos nas mãos até 10 utilizando o sistema decimal, já utilizando o sistema binário é possível contar até 1023, já utilizando o sistema ternário uma pessoa com destreza suficiente nas mãos para conseguir controlar os dedos individualmente para 3 posições (fechado, aberto e entreaberto) conseguiria contar teoricamente até 59,048 utilizando as duas mãos.
